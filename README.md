# TaskManager  
Aplicación Android desarrollada en Java utilizando **Room Database**, **RecyclerView** y **Android Studio**, diseñada para gestionar **tareas**, **categorías** y **notas** (solo Actividades 1 y 2).

---

## Descripción del proyecto
TaskManager es una aplicación móvil que permite:

### **Actividad 1 – Gestión de Tareas**
- Registrar tareas (título, descripción, fecha).
- Mostrar una lista de tareas en un RecyclerView.
- Editar o eliminar tareas desde un formulario.
- Guardarlas localmente mediante **Room Database (Entidad: Task)**.

### **Actividad 2 – Categorías y Notas**
- Crear categorías personalizadas.
- Listarlas en un RecyclerView.
- Registrar notas dentro de cada categoría.
- Utilizar relaciones **Category → Notes** con Room.
- Mostrar notas relacionadas con cada categoría (One-to-Many).

> Nota: Este repositorio **solo incluye Actividad 1 y Actividad 2** tal como lo solicitó la evidencia.  
> La Actividad 3 no está incluida aún.

---
